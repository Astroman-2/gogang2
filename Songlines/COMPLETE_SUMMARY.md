# Songlines Project Summary

## Overview
Songlines is a high-fidelity 'Eldritch Archive' themed web dashboard. It features a continuous scroll lyric engine, dynamic image transitions, and an integrated media player.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Custom Variables), JavaScript (ES6+)
- **Theme**: Dark panel aesthetics with gold accents (#D4AF37)
- **Typography**: Calligraphy-style headers and refined serif body text.

## Features
- Responsive media layout.
- 10-chapter image gallery.
- Background video integration.
